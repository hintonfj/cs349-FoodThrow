FOOD THROW
----------
Hello and welcome to Food Throw!
Once you run the jar you will be presented with three game mode options. 

Free Play
* Paint freely, with no objective,.
* User is able to save their drawings as a png through the save button in top right hand corner. 

Speed Paint
* A timed accuracy challenge.
* The user is presented with an image that pops up alongside your canvas.
* A circular countdown timer shows how much time is left.
* When time is up (or you press Finish!), your artwork is compared pixel-by-pixel to the original and the user is given an accuracy rating.

Fill Wall
* A filling challenge.
* The user is presented with an a shape on the wall that must be completely filled with paint.
* You have 100 shots total to fill the shape with
* Win by filling the shape before running out, lose if shots reach zero while color remains.

----------
Known bugs
* In Fill Paint, shot counter is a count behind when game finishes.
